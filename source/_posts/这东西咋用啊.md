总算搭起来了，总结一下步骤

1. 按照 [hexo readme](https://github.com/hexojs/hexo) 的步骤初始化一个文件夹，命名为 `username.github.io`

2. 创建文件 `.github/workflows/pages.yml`，内容照着[这里](https://hexo.io/docs/github-pages.html)写

3. push 到 GitHub 仓库

4. 到 GitHub 仓库的 `Settings > Pages > Build and deployment` 把 Branch 改为 `gh-page`
