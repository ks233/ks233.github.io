# 乐器与音色

本文内容以[好和弦](https://nicechord.com/)直播为框架，加入了一些自己查找的图片与资料，涵盖了大部分编曲中常用的乐器和音色。

## 键盘 Keyboard

### 三角钢琴 Grand Piano

比较亮的音色，适合绝大多数场景。

![有钱人家的钢琴](piano.png)

推荐音源：

- **Keyscape**：采样了 Yamaha C7 三角钢琴，也有立式钢琴音色。
- **Pianoteq**：物理模拟而非采样的音色。力度曲线可调，也就是可以调整 MIDI 输入的力度和实际发出声音的力度的关系。也可以模拟声音远近。

#### 常见调整

有些音源的预设很烂，可以尝试关闭自带效果器、调短 Release Time 使音色更干净。

#### 古典录音

古典乐通常会在距离比较远的地方录音，而且受到录音技术的限制，音色会与流行钢琴有所不同。物理模拟音色 Pianoteq 可以比较好地实现这种效果。

### 立式钢琴 Upright Piano

声音比三角钢琴糙一些，适合比较家庭、比较温馨的氛围感。

![一般家庭的钢琴](upright_piano.png)

推荐音源：NI 的 **UNA CORDA**。名字的原意是单弦，或者弱音踏板。可以发出比较闷的声音，经常在电影配乐中出现，适合一些抑郁、恐怖的场景。

### 酒吧钢琴 Honky-tonk Piano

指酒馆里那种比较破旧、调音不太准的钢琴，适合弹 Ragtime 之类休闲、诙谐的风格，这类音乐用很准的三角琴弹反而不对味。

### 电钢琴 Electric Grand Piano

用真钢琴的机械结构发声，然后用类似电吉他的方式放大声音。

在上世纪比较流行，经典型号有 Yamaha CP70、CP80 等。

听起来像是比较劣质的合成器音色，现在不太常用了。

![Yamaha CP-80](cp80.jpeg)

### Rhodes Piano

最有代表性的电子琴音色。

与电钢琴不同之处在于琴锤敲击的不是琴弦，而是敲金属片，然后拾音、放大。

与真钢琴相比，Rhodes 更适合当 Bass，钢琴更适合弹一些琶音和华彩乐段。

常见自带效果：

- **Chorus**：多个音叠在一起的效果
- **Auto Pan**：左右声道来回移动的效果

Rhodes 是发明者的名字（Fender Rhodes）。

![Rhodes Piano](rhodes.jpg)

### Wurlitzer

另一种电子琴。听起来比较像合成器的方波。

![感觉这几个电子琴长得都差不多啊](Wurlitzer.jpg)



### FM Electric Piano

一般指与 Yamaha DX7 类似的音色，有 90 年代西方慢流行歌的感觉。

FM 就是收音机那个 FM，这种音色是纯合成的，现实中没有对应的乐器。

通常会加很多 Reverb 和 Chorus 效果。

![Yamaha DX7](yamaha_dx7_angle2.jpg)



### 大键琴 Harpsichord

巴洛克时期的键盘乐器，有两排键盘，没有踏板，听起来很适合弹巴赫。

![Harpsichord](Harpsichord.jpeg)

### Clavinet

声音比较像电吉他的键盘乐器，可以加一些电吉他效果器。

不插电的吉他。

### 电吉他 Electric Guitar

![Gibson Les Paul Standard](Electric_Guitar.jpg)

- **Clean**：清音，没有失真效果的电吉他。

## 贝斯 Bass

### 低音提琴 Acoustic Bass

Acoustic Bass = Double Bass = Upright Bass

拨奏的低音提琴。

![演奏提琴不一定要用琴弓](double-bass.jpg)

### 电贝斯 Electric Bass

演奏方法：

- **Fingered**：用手指拨奏
- **Picked**：用拨片演奏，声音比较亮
- **Slap**：用大拇指用力敲击琴弦

![soyo 自愿演奏春日影](Electric_Bass.png)

也有无品贝斯（Fretless Bass），可以演奏滑音。

贝斯的四根弦分别是 E1 A1 D2 G2，音域是 E1 ~ D#4，**很多编曲软件会移高八度**，注意听声音区分。

### 合成器贝斯 Synth Bass

在编曲软件中经常和上面两个放在一起，但实际上是完全不同的东西。

合成器贝斯只是某种比较适合当 bass 用的合成音色，没有对应的乐器。**如果要做流行乐而不是电子乐，不建议用这种合成音色。**

## 弦乐 Strings

![四种提琴](strings.jpg)

### 乐器音高

| 乐器                 | 空弦音高    | 说明           |
| -------------------- | ----------- | -------------- |
| 小提琴 Violin        | G3 D4 A4 E5 |                |
| 中提琴 Viola         | C3 G3 D4 A4 | 比小提琴低五度 |
| 大提琴 Cello         | C2 G2 D3 A3 | 比中提琴低八度 |
| 低音提琴 Double Bass | E1 A1 D2 G2 | 和电贝斯相同   |

知道了空弦的音高，就能知道如何正确地编写弦乐。以小提琴为例：

1. 小提琴不能拉低于 G3 的音。
2. 小提琴最多同时发出四个音。
3. **要写四个弦可以各自发出的声音**，比如 G3 + C4 就是单个小提琴不可能发出的音。

### 编曲技巧

编曲软件里一般有**合奏（Ensemble/Section）**和**独奏（solo）**音色，区别很明显。

独奏很难用编曲软件做好，最好直接录，实在不行可以用 **Samplemodeling** 的 Violin Solo。

建议不用琴键力度控制小提琴力度，而是用一个独立的 MIDI 控制器控制，通常是 CC11 控制。

在写谱的时候会把和弦的不同的音分到不同的乐器，但在编曲的时候，可以把主旋律分一轨，其它伴奏性质的音分一轨。

### 常见演奏法（Articulation）

- **Sustain**：普通的长音
- **Staccato**：断奏或跳音
- **Pizzicato**：拨奏，活泼的感觉
- **Tremolo**：震音（音量变化）
- **Vibrato**：揉弦（音高变化）

### 音色库常见缩写

- **VB**：揉弦（Vibrato）
- **NV**：不揉弦（Non-Vibrato）
- **RR**：每次按同一个键触发不同采样（Round Robin）
- **Ni**：渐弱最低可以到无声

### Synth Strings 不是弦乐

与 Synth Bass 同理，不要把 Synth Strings 当做弦乐，它是合成器音色。

## 铜管 Brass

### 主要乐器

![常见的铜管乐器](brass.jpg)

- 小号 **Trumpet**
- 圆号 **French Horn**
- 长号 **Trombone**
- 大号 **Tuba**

### 常见演奏法

- **Sustain**
- **Staccato**
- **Vibrato / Non-Vibrato**
- **fp cresc**：先强后弱，然后再渐强
- **Fall / Doit**：下滑、上滑
- **Rip**：从下往上滑到本音

### 其它知识

小号可以用弱音器改变音色。

和弦乐音色类似，管乐音色同样有 Solo 和 Section 的区分，一般用 CC1 或者 CC11 控制力度。

Synth Brass 不是 Brass。

## 合成器 Synthesizer

合成器的取名规律：

- **用听起来像的乐器命名**，比如 Brass / Strings / Guitar
- **Lead**：穿透力强，适合用单音作为主旋律的音色
- **Pad**：适合垫在后面的音色。
- **Pluck**：很像拨弦声音的音色。
- **5th**：按一个键，实际上会产生五度音。

其它关键词：

- **Poly / Mono**：多音 / 单音
- **Glide / Portamento**：音和音之间衔接处的滑音速度
